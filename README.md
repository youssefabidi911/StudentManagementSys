# 📚 Student Management System

A comprehensive student management system built with Python and MySQL, featuring grade management with weighted coefficients, attendance tracking, and user authentication.

## ✨ Features

- **User Authentication** - Secure login for admins and teachers
- **Student Management** - Add, edit, and search student records
- **Grade Management** - Track grades with weighted coefficients per subject
- **Attendance Tracking** - Mark student attendance (Present/Absent/Late)
- **Statistics Dashboard** - View key metrics and performance analytics
- **User Management** - Admin-only interface for managing system users
- **Search Functionality** - Search students by name, ID, or grade
- **Reports** - Generate performance reports and statistics
